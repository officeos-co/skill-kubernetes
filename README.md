# Kubernetes

Manage Kubernetes clusters, workloads, services, config, nodes, and jobs via the Kubernetes REST API (kubectl parity).

## Categories

- Cloud Infrastructure

## Credentials

| Key | Label | Kind |
|-----|-------|------|
| `api_url` | API Server URL | text |
| `token` | Bearer Token | password |
| `namespace` | Default Namespace | text |

## Installation

This skill is included in [EnterpriseAgentOS](https://officeos.co) and requires no separate installation.
